# 📄 RPA Multiple PDF Data Extraction (UiPath)

---

## 🚀 Project Overview
This project is a **PDF Data Extraction Automation System built using Robotic Process Automation (RPA)** with UiPath.  
It automates the process of extracting data from multiple PDF files and processing it efficiently.

The project demonstrates how RPA can handle large volumes of unstructured or semi-structured data from PDFs, reducing manual effort and improving accuracy.

---

## 🎯 Objectives
- Automate extraction of data from multiple PDF files  
- Reduce manual data entry effort  
- Improve accuracy and speed of data processing  
- Handle bulk document processing efficiently  
- Demonstrate real-world RPA use case  

---

## ⚙️ Key Features
- 📂 Processes multiple PDF files automatically  
- 🔍 Extracts required data from PDFs  
- 📊 Converts unstructured data into structured format  
- 📤 Stores extracted data for further use  
- 🔁 Handles repetitive document processing  
- ⚡ Fast and accurate automation  

---

## 🛠️ Technologies Used
- **UiPath Studio**  
- **XAML Workflows**  
- **.NET Framework**  
- **PDF Activities (Read PDF / OCR / Data Extraction)**

## 🔄 Workflow Explanation
1. The automation starts from **Main.xaml**  
2. Reads multiple PDF files from a specified folder  
3. Extracts data using PDF/OCR activities  
4. Processes extracted information based on logic  
5. Converts data into structured format (table/text)  
6. Stores or outputs the processed data
